# AIHS Attendance Frontend

Single-page attendance system for Askari Institute of Health Sciences (AIHS), PDS24 Semester 3 Pharm-D.

## Live URL
https://forfacebook687-sketch.github.io/aihs-attendance-frontend

## Features
- Mark Attendance with student cards (Present/Absent toggle)
- Today's subjects auto-loaded from timetable
- DAR (Daily Activity Report)
- Defaulter list (below 75%)
- Student summary with per-subject percentages
- AI Scanner — photo of paper register auto-marks attendance
- Export CSV
- Admin panel — teacher management, subject reassignment, timetable switching, API key

## Setup
1. Deploy backend to Railway first
2. Update `API_URL` in `index.html` to your Railway URL
3. Enable GitHub Pages: Settings > Pages > Branch: main
